# image-filter-module
Image filtering module based on chrome extention

# 유해 컨텐츠 필터링 모듈

## Note
유해 컨텐츠(음란 이미지, 도박 배너)를 필터링하는 크롬 브라우저 모듈입니다.

## Dependency
Server
- python 3.7
    - flask 1.0.2
      - flask_cors 3.0.7
    - tensorflow 1.13.1
    - numpy 1.16.2
    - PIL 5.4.1
    - cv2 4.1.0

Client
- Javascript
	  - chrome extension 2.0 API
